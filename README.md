# 明外史 (MingWaiShi)

## 📖 项目简介

**明外史**是一个致力于收集、整理和补充明朝历史人物传记的学术项目。本项目汇集了大量未被《明史》正式收录或记载不全的明朝人物传记资料，旨在为明史研究提供更全面的史料补充。

## 🎯 项目目标

- 补充《明史》中缺失或记载简略的人物传记
- 收集整理明朝各类人物的详细生平资料
- 为明史研究者和历史爱好者提供参考资料
- 保存和传承明朝历史文化

## 📁 项目结构

```
MingWaiShi/
├── 参考书目/          # 历史文献和参考资料
│   └── 中外旧约章汇编 第1册 (王铁崖).pdf
│
└── 所补传记/          # 明朝人物传记文档
    ├── 补明史系列传记
    ├── 各类人物传记
    └── 历史评传资料
```

## 📝 收录内容

### 人物类别

本项目收录的人物传记涵盖明朝各个群体：

- **忠义人物**：如张奇策、郑人怿、许仪后、郭国安等
- **党人传**：如汪文言、王象春、賀烺等
- **儒林人物**：如卢宁等
- **女性人物**：如唐赛儿、韩桂兰、高妙莹、邢慈静、文俶等
- **军事人物**：如邵荣、赵普胜等
- **起义人物**：如林凤、黄萧养等
- **外交与外族**：如利玛窦、哈立麻、女直传等
- **其他重要人物**：各类官员、学者、名士等

### 文档类型

- **传记原文**：.docx, .doc, .txt 格式
- **加引版本**：标注引用出处的版本
- **评传资料**：.pdf 格式的学术评传
- **历史文献**：参考书目和史料汇编

## 🔍 当前进度

项目目前已收录：
- **传记文档**：约 60+ 份人物传记
- **参考文献**：历史文献资料
- **工作计划**：补明史计划（2025-11-24）

## 📚 主要特色

1. **资料详实**：提供多个版本的传记文本，包括加引版本（标注史料出处）
2. **覆盖广泛**：涵盖明朝各个阶层和群体的人物
3. **学术严谨**：参考多种历史文献，注重史料考证
4. **持续更新**：项目持续收集和整理新的传记资料

## 🤝 如何使用

1. **浏览传记**：进入`所补传记`文件夹查看感兴趣的人物传记
2. **查阅文献**：`参考书目`文件夹提供历史文献支持
3. **学术引用**：使用加引版本可以快速定位史料来源

## 📖 文件命名说明

- **补明史·xxx传**：《补明史》体例的传记
- **xxx传（加引版）**：标注史料引用出处的版本
- **xxx(数字)**：同一人物的不同版本或修订版
